AfroPhyto Documentation

AfroPhyto provides researchers, educators, and enthusiasts with detailed, curated information on African medicinal plants and their bioactive compounds. This documentation guides you through efficient searching, exploring plant and phytochemical data, contributing new entries.

Introduction

AfroPhyto is a curated, open-access database of African medicinal plants and their bioactive phytochemicals. It provides researchers, students, and practitioners with detailed information about plant species, chemical compounds and biological activities.

Searching

You can search for plants using either common names or scientific names. Search is case-insensitive and supports partial matches.

  • Use the search bar on the homepage.
  • If no results are found, a friendly message will display: "No plants found matching your search query."
  • Example: Searching for "Mango" will return all plants with Mango in the common or scientific name.

Browsing Plants

Browse the complete catalog using pagination:

  • Efficient navigation through multiple pages of entries.
  • Quick overview of plant names and key details.
  • Access to individual plant pages with full phytochemical data.

Data Structure

Each plant entry contains the following fields:

  • Common Name: Popularly used name(s) of the plant.
  • Scientific Name: Standard binomial nomenclature.
  • Family: Taxonomic family.
  • Region / Origin: Geographic distribution.
  • References: Literature or ethnobotanical sources.

Phytochemicals

For each plant, you can view associated bioactive compounds:

  • Chemical Name: Official chemical name.
  • SMILES: Simplified Molecular Input Line Entry System.
  • Biological Activity: Known medicinal properties or pharmacological actions.
  • References: Sources of experimental evidence.

Contributing

Contributions are welcomed from researchers and experts. To submit new plant or phytochemical data:

  1. Navigate to the Contribute/Submit Data page.
  2. Complete the submission form accurately.
  3. Attach relevant images or references.
  4. Data is reviewed by the AfroPhyto team before publication.

Citing AfroPhyto

If you use AfroPhyto in research or publications, please cite it as follows:

AfroPhyto: An open atlas of bioactive phytochemicals from African plants. 
Journal Name, Volume(Issue), pages. https://doi.org/XXXX

AfroPhyto. (2026). An open atlas of bioactive phytochemicals from African plants.
